# C_churrovis

Data for the anaerobic gut fungus *Caecomyces churrovis* (JGI MycoCosm genome portal code: `Caecom1`).

## Contents

### `C_churrovis_genome/`
Reference genome and gene catalog from JGI MycoCosm (annotation release 2017-12-13).

| File | Description |
| --- | --- |
| `Caecom1_AssemblyScaffolds.fasta` | Nucleotide scaffold assembly (FASTA, ~168 MB). |
| `Caecom1_GeneCatalog_20171213.gff3` | Gene models / annotations in GFF3 format. Attributes include `ID`, `Name`, `portal_id`, `product_name`, `proteinId`, `transcriptId`. |
| `Caecom1_GeneCatalog_proteins_20171213.aa.fasta.gz` | Predicted protein sequences (amino acid FASTA, gzipped). |
